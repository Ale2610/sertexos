Welcome to SertexOS site

SertexOS is a simple no-touch operating system for ComputerCraft 1.63 - 1.64

Features

Password system crypted by SHA256 LuaIDE for dev
Ice-Browser for exploring files
SertexGPS if you lost your house you can try SertexGPS

How to install

To install SertexOS

run "pastebin run 31AUQX7g" without quotes

Warning: You need advanced computer to run SertexOS
